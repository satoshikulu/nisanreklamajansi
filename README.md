# Kulu Nisan Reklam Ajansı - Web Sitesi

Modern, responsive tek sayfa web sitesi.

## 🚀 Teknolojiler

- **React** - UI framework
- **Vite** - Build tool
- **Tailwind CSS** - Styling
- **Framer Motion** - Animasyonlar
- **Lucide React** - İkonlar
- **React Router DOM** - Navigasyon

## 📦 Kurulum

```bash
# Bağımlılıkları yükle
npm install

# Development server'ı başlat
npm run dev

# Production build al
npm run build

# Production preview
npm run preview
```

## 📁 Proje Yapısı

```
src/
├── components/
│   ├── layout/
│   │   ├── Header.jsx      # Sticky header + mobil menü
│   │   ├── Footer.jsx      # Footer bileşeni
│   │   └── Layout.jsx      # Ana layout wrapper
│   └── sections/
│       ├── Hero.jsx        # Ana banner bölümü
│       ├── Services.jsx    # Hizmetler bölümü
│       ├── Gallery.jsx     # Galeri/Portfolio
│       └── Contact.jsx     # İletişim formu + bilgiler
├── pages/
│   └── Home.jsx            # Ana sayfa
├── App.jsx                 # Ana uygulama
└── main.jsx                # Giriş noktası
```
